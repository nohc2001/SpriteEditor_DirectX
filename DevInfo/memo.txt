InsideCode의 실행방법은 다양하다
하지만 실행에는 3가지 과정이 생기는데,
1. 코드의 텍스트를 모두 읽어들이는 과정
2. 코드의 텍스트를 코드데이터로 변환.
3. 코드데이터들을 컴파일.

실행 과정 많은 부분에서 사용자가 수정을 할 수 있게 해야 한다.
예를 들어 코드의 텍스트 부분에서 1000번째 글자를 0에서 1로 바꾸거나,
3번째 코드 데이터인 반복문을 없애는 것 같은 수정을 할 수 있겠다.
그러기 위해 어떤 함수를 준비해야 하는가?

1. 코드 파일 불러와서 단어 추출하는 함수
2. 데이터의 코드라인 데이터를 불러와 단어 추출하는 함수.
3. 해석하여 코드 데이터를 만드는 함수.
4. 코드데이터를 컴파일하는 함수 > 부분컴파일도 목표에 있지만 아직임.
> 4 번 과정이 진행되는 중에는 ECS로 코드를 돌릴 수 없다.
